개인정보처리방침

최종 갱신: 2026-05-07

1. 수집하는 정보

  • Chess.com 또는 Lichess 닉네임 (게임 데이터 조회 목적)
  • 분석 결과 — 정확도, 수 분류, 블런더 / 놓친 메이트 풀이 항목
  • 사용자가 명시적으로 저장한 게임 / 복기 항목
  • 익명 사용 통계 (Vercel Analytics 페이지뷰)

이메일, 비밀번호, 결제 정보 등 민감 정보는 수집하지 않습니다. 별도 회원 가입 절차도 없습니다.

2. 이용 목적

  • Chess.com / Lichess 공개 API에서 본인 게임 조회
  • Stockfish 엔진 기반 분석 결과 표시
  • 분석 캐시 저장 — 재방문 시 재분석 비용 절감
  • 자동 블런더 / 놓친 메이트 풀이 수집
  • 승률 / 오프닝 / 시간대 통계 산출

3. 보관 기간 및 삭제

  • 앱 내 "로그아웃"을 누르면 닉네임 정보가 초기화됩니다 (단, 저장된 복기 / 게임은 명시적으로 삭제하지 않으면 유지)
  • 본인 데이터 전체 삭제를 원하시면 아래 연락처로 요청해 주세요 — 즉시 처리합니다

4. 제3자 위탁

  • Supabase — 분석 결과 / 복기 / 저장 게임의 데이터베이스 보관
  • Vercel — 호스팅 및 익명 페이지뷰 분석
  • Google Gemini API — 사용자가 "AI에게 묻기" 기능을 명시적으로 실행할 때에 한해 해당 게임의 수순 데이터 전송
  • Chess.com / Lichess 공개 API — 사용자가 입력한 닉네임으로 본인 공개 게임 조회

5. 사용자 권리

  • 수집된 본인 데이터의 열람 / 삭제 요청 가능
  • 요청은 아래 연락처를 통해 가능하며, 즉시 처리됩니다
