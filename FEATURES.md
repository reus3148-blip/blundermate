# ♟️ Blundermate (Chess Review App) 기능 명세서

현재 프로젝트에 구현되어 있는 주요 기능 목록입니다.

## 1. 게임 불러오기 및 기보 분석 (PGN & API)
- **수동 PGN 입력**: 사용자가 직접 PGN 텍스트를 복사하여 붙여넣고 분석을 시작할 수 있습니다.
- **유연한 기보 파싱**: 정규 PGN 형식이 아니거나 숫자/기호가 누락된 텍스트라도 자체적으로 복구 및 정규화하여 불러옵니다.
- **Chess.com API 연동**: 사용자 이름을 입력하면 Chess.com에서 최근 10개의 게임을 가져와 목록으로 보여주고, 선택 시 바로 분석할 수 있습니다.

## 2. 엔진 분석 시스템 (Stockfish)
- **로컬 Web Worker 엔진**: 브라우저 내부에서 `stockfish-18-lite-single.js`를 구동하여 빠르고 독립적인 기보 분석을 수행합니다.
- **자동 수(Move) 평가**: 모든 수를 순차적으로 분석하며, 이전 수와의 Centipawn 손실(CPL) 및 승률(Win%) 변동을 계산하여 수를 평가합니다.
- **평가 등급 분류**: Best, Good, Inaccuracy, Mistake, Blunder, Missed Win 으로 분류하여 색상과 함께 표시합니다.

## 3. 인터랙티브 체스 보드 (Chessground)
- **실시간 보드 탐색**: 화살표 키나 버튼을 이용해 기보를 앞뒤로 이동하며 탐색할 수 있습니다.
- **화살표 시각화**: 엔진이 추천하는 최선의 수, 혹은 Blunder/Mistake 상황에서 놓쳤던 최선의 수를 체스판 위에 화살표로 그려서 보여줍니다.

## 4. 탐색(Exploration) 및 시뮬레이션 모드
- **자유 탐색 모드**: 분석 중인 보드에서 기물을 직접 움직여 다른 라인(가정의 수)을 실시간 엔진 평가와 함께 탐색할 수 있습니다.
- **시뮬레이션 모드**: 엔진이 추천하는 여러 갈래의 라인(MultiPV) 중 하나를 클릭하면, 해당 라인이 체스판 위에서 자동으로 시뮬레이션 됩니다.

## 5. 생성형 AI 해설 (Gemini API 연동)
- **자연어 해설 기능**: Gemini 1.5 Flash API를 활용하여, 현재 국면에서의 최선의 수의 이유나 사용자가 둔 Blunder/Mistake의 전술적/포지션적 문제점을 체스 그랜드마스터 관점에서 2~3문장으로 설명해 줍니다.
- **API 키 로컬 저장**: 사용자로부터 입력받은 API 키를 브라우저 `localStorage`에 안전하게 저장하고 관리합니다.

## 6. 오답 노트 및 저장소 (My Vault & Saved Games)
- **개별 수 저장 (My Vault)**: 특정 수(특히 Blunder나 Mistake) 상황을 분류(카테고리) 및 메모와 함께 보관함에 저장할 수 있습니다.
- **연습 모드 (Practice Mode)**: My Vault에 저장된 틀린 국면을 다시 띄워놓고, 엔진이 제안했던 '최선의 수(Best Move)'를 직접 찾아보는 복습 기능을 제공합니다.
- **전체 게임 저장 (Saved Games)**: 분석이 완료된 게임 전체(PGN)를 제목과 메모를 덧붙여 로컬에 영구 저장하고, 나중에 다시 불러올 수 있습니다.

## 7. UI/UX 및 최적화
- **반응형 디자인**: Flexbox와 CSS 변수를 활용한 다크 모드 기반의 모던 웹 디자인을 제공하며, 모바일 화면에서는 보드 크기와 레이아웃이 알맞게 최적화(리사이징) 됩니다.
- **상태 표시 기능**: 상단에 평가 바(Score)를 직관적으로 띄워주고, 엔진 상태(Loading, Ready 등)를 태그 형식으로 보여줍니다.
- **모달 창 기반 조작**: 게임 저장, 수 저장, 선택 메뉴 등 부가 기능들이 깔끔한 모달 오버레이 형태로 제공됩니다.
