홈 화면 레이아웃을 재정렬하고 "수 입력" 링크 이름을 변경해줘.

## 요구사항

### 1. 홈 화면 구조 변경
현재 "다른 유저 검색 / PGN / FEN / 직접 두기" 링크 두 개가 게임 리스트 **아래**에 있음. 이를 프로필 박스 **아래 + 게임 리스트 위**로 이동.

변경 후 구조:
```
[상단 헤더 — 로고]

[프로필 박스]
 bywxx
 ─────────
 1129 / 686 / 364

  다른 유저 검색   수 입력    ← 프로필 아래로 이동

[게임 리스트]
  vs ihsantoker ...
  vs El90TR ...
  ...

[복기 | 저장된 게임]

       피드백  설정
```

### 2. 링크 텍스트 변경
- "PGN / FEN / 직접 두기" → **"수 입력"**
- "다른 유저 검색" → 그대로 유지
- 링크 스타일(언더라인 텍스트) 자체는 유지

### 3. strings.js 수정
- 기존 "PGN / FEN / 직접 두기" 관련 키의 value를 "수 입력"으로 변경
- ko/en 양쪽 수정:
  - ko: "수 입력"
  - en: "Enter Moves" (또는 기존에 맞는 표현 유지)
- 키 이름 자체는 바꿀 필요 없음 (value만 변경)
- 해당 링크가 진입하는 화면의 타이틀은 이미 "수 입력"으로 되어있어서 자동으로 일관성 유지됨 — 별도 변경 불필요

### 4. 간격 체계 재정리
```
[프로필 박스]
    ↕ 중간 간격
[링크 2개]
    ↕ 좁은 간격 (같은 "분석 진입" 묶음으로 묶기)
[게임 리스트]
    ↕ 넓은 간격 (섹션 구분)
[복기 | 저장된 게임 페어]
    ↕ 기본/약간 넓은 간격
[피드백 / 설정 푸터]
```

- 링크와 게임 리스트는 **같은 단락**처럼 묶음 → 사이 간격 좁게
- 게임 리스트 ↔ 복기/저장된 게임 페어: 섹션 구분 위해 넓게
- 기존 CSS 변수 활용, 하드코딩 금지

### 5. 확인사항
- 게임 없는 신규 유저 상태에서도 레이아웃 자연스러운지 (링크 위치 이상 없어야 함)
- 다른 유저 검색 중(viewingUserId) 상태에서도 링크 두 개 동작 정상 — 다른 유저 검색 / 수 입력 둘 다 항상 접근 가능해야 함
- 데스크톱(max-width 420px 프레임) / 모바일 양쪽
- 링크 이름 변경 후 진입 화면과 일관성 확인 ("수 입력" 링크 → "수 입력" 타이틀 화면)

### 6. 디자인 시스템 준수 (CLAUDE.md)
- CSS 변수만 사용
- 파랑/인디고 금지
- 앰버 `--ac`만 포인트
- 다크 전용
